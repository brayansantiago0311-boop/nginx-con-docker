# Nginx con Docker Compose

Práctica académica donde se crea un servidor web Nginx ejecutándose localmente utilizando Docker Compose.

## 📌 Pasos realizados

1. Creación de carpeta `src` con archivo `index.html`
2. Configuración de `docker-compose.yml`
3. Levantamiento del contenedor con:
   ```bash
   docker compose up -d
